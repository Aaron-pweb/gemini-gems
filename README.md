# Gemini Gems

Reusable Gemini Gem instruction files for focused AI assistants.

## Gems

- [Nexus](./nexus.md): Executive productivity assistant for Calendar, Gmail, and Tasks workflows.
- [Social Media Manager](./social-media-manager.md): Turns developer updates into platform-specific posts.
- [Terminal Geek](./terminal-geek.md): Diagnoses terminal errors, build failures, and stack traces.

## Usage

Open a `.md` file, copy the instructions into a Gemini Gem, then connect any required tools or extensions for that gem. Keep each gem focused on one job, with explicit inputs, decision rules, and output formats.

## Maintenance

- Keep instructions specific and testable.
- Add confirmation requirements before any external side effect.
- Prefer structured output so responses are easy to scan.
- Update examples when workflows change.
